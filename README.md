# My_DIR
## Project Overview:
Implementation the dir command behaviour with three options:
### 1. dir /Q 
display the owner of the file.
 
### 2.dir /L
diplay files name  in lower case.

###3.dir /S
Displays files in specified directory and all subdirectories.
